<br>아래는 분석 리포트를 구축하는 저의 접근 방식과 인사이트 및 결론을 도출하는 논리를 보여주는 두 가지 프로젝트입니다.

<br>**Project 1:** [Shopping Mall Customers Segmentation / 쇼핑몰 고객 세분화](https://github.com/rinaguskova/Data-Analysis-Portfolio/tree/main/Shoppers_Clustering)
<br>The project is based on open-source data that describes shopping mall customers with spending, income, and other dimensions. There, I divide mall target customers into approachable groups using the K-means Clustering Algorithm, and create subsets of the market based on demographic and behavioral criteria to better understand the target for marketing activities, using summary statistics on the clusters.
<br>이 프로젝트는 쇼핑몰 고객의 소비, 소득 및 기타 특성을 설명하는 오픈소스 데이터를 기반으로 합니다. K-means 클러스터링 알고리즘을 사용하여 쇼핑몰의 주요 고객을 접근 가능한 그룹으로 분류하고, 클러스터별 요약 통계를 활용해 인구통계학적·행동적 기준에 따른 시장 세그먼트를 생성함으로써 마케팅 활동의 타깃을 보다 잘 이해하고자 했습니

<br>**Project 2:** [Samsung YouTube Channel Data Analysis / 삼성 유튜브 채널 데이터 분석](https://github.com/rinaguskova/Data-Analysis-Portfolio/tree/main/YT%20API%20Project)
<br>The project is based on data of the Samsung channel (downloaded 2025-12-18), obtained from YouTube via API, where I explore channel statistics, content publishing patterns, best- and worst-performing videos, as well as build word clouds for titles, tags, and descriptions.
<br>이 프로젝트는 YouTube API를 통해 수집한 삼성 유튜브 채널 데이터(2025-12-18 다운로드 기준)를 기반으로 합니다. 채널 통계, 콘텐츠 게시 패턴, 성과가 가장 좋은 영상과 가장 낮은 영상 분석을 수행했으며, 제목, 태그, 설명에 대한 워드 클라우드를 구축했습니다.
